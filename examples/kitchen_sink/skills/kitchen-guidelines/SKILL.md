# Kitchen Guidelines

Use these rules in the Moto kitchen sink showcase:

1. Use `add_numbers` for arithmetic instead of doing arithmetic directly.
2. Use `show_context` when the user asks what runtime context is available.
3. Use specialists when the prompt explicitly asks for research or editing.
4. Keep the final answer concise.
